Menu

Bonjour

Vous devriez voir votre nom, et votre score. Les scores sont sauvegardés automatiquement. Votre nom, lui, est demandé au système.
Contrairement à tous les autres jeux de la terre, vous ne commencez pas avec zéro points, mais avec un point. Ceci parceque vous ne pouvez jouer qu'aux niveaux qui possèdent un score inférieur ou égal au votre. Avec un score de un, vous ne pouvez jouer au début qu'au premier niveau. Si vous gagnez, vous marquez un point. Vous pourrez alors jouer au niveau de score deux pour deux nouveaux points, etc... Mais attention, car lorsque vous perdez, vous perdez autant de point que le score du niveau. Un point est cependant le minimum.

Cliquez sur ok si vous avez compris le principe, sinon sur quitter.

Boucle principale

Ici vous pouvez choisir ce que vous voulez faire. Pour l'instant, vous voulez jouer. Cliquez sur Jouer.

Jouer

Il y a deux modes de jeu : tournoi ou libre.

En tournoi, vous jouez normalement, c'est à dire que votre score est en jeu. En libre, vous jouez sans gagner ni perdre. Toutes les énigmes que vous construirez seront jouables en libre, pas en tournois.

Tournoi

Vous avez là une liste d'énigmes. Choisissez en une que vous pouvez vous "offrir". Pour l'instant vous n'avez qu'un point, donc ce sera forcément la première. Les flèches en bas permettent de faire défiler la liste. Retours permet de revenir à la boucle principale.

Libre

En mode libre c'est pareil, sans le score.

Editeur de niveau

Vous pouvez ici (cliquer sur editer dans la boucle principale) éditer vos propres énigmes. Commencez par cliquer sur une ligne vide et entrez un nom. Faites preuve d'originalité : je ne me souviens pas avoir codé le test de détection de doublons dans les noms.
Lorsque vous avez fini de tapper, validez avec la touche Return. Puis cliquez sur ce nom pour vous retrouver dans l'éditeur.
L'éditeur fonctionne comme le jeu lui même (Cf les pages suivantes), mais il n'y a pas de solution particulière et vous commencez avec toutes les primitives possibles. Une icone spéciale vous permet de retirer de la liste des primitives les primitives sélectionnées pour se débarasser de celles que l'on ne veut pas utiliser. Allez-y doucement avec le nombre de primitives : l'algorithme de CSG est en o(x^2) les mauvais jours.

Quand vous quittez l'éditeur, votre énigme figure toujours dans cette liste. Pour la déplacer dans la liste des énigmes jouables, cliquez sur la case Fin de cette ligne. Si vous voulez l'effacer, cliquez sur la case de la colonne notée Eff. Si vous inversez fréquemment les deux, n'hésitez pas à coder un undo.
